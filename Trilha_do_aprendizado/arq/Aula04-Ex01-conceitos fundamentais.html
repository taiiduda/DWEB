HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee para a publicação de conteúdo em hiper-texto, tais como texto, imagens, áudio e vídeo, que podem ser vivinculados através de hyperlinks.

TAGS

As tags definem ao navegador a marcação do conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano

As tags podem ser atributos, ou seja, são parâmentros que definem algum tipo de informação extra ao navegador.
