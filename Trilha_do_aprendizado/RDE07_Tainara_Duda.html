Branches:

São ramificações de um projeto de softwareou domumento e tem como objetivo
o desenvolvimento paralelo de :

  • Novas Funcionalidades;
  • Correção de problemas;
  • Alteração de código.
Branch Master ou Main:
Braço principal de um projeto em um repositório
Branch Secundária:
Braço secundário de um projeto em um repositório

Para juntar as Branches depois de ter certeza das alterações de update, é feito um
Merge(ou mesclagem) das branches.

Também é possivel ter uma branch terciária dentro de uma branch secundária, ou quantas
forem necessárias.

Hospedagem:

O github Pages oferece hospedagem gratuíta de sites. Basta criar uma conta em um
respositório público e a URL será: https://nomeusuario.github.io/repositório
